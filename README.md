# DMN Workshop Labs

This project holds the resulting dmn models and projects of the guided exercises in the RHPAM/RHDM DMN workshop.

* DMN for the use cases:
  * Policy Price
  * Vacation Days
  * Call Center
* Java client for:
  * Vacation Days
  * Call Center

Tested on: RHPAM 7.10.0
